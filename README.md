# 📚 ZYPP – Book Price Comparison Platform

Compare book prices across Amazon, Flipkart, and Bookswagon to find the best deal before purchasing.

## 📖 About the Project

ZYPP is a PHP & MySQL based web application that helps users compare book prices across multiple online retailers in one place.

Instead of searching each shopping website individually, users can view prices from Amazon, Flipkart, and Bookswagon on a single comparison page and directly visit the retailer offering their preferred deal.

The platform provides an easy and convenient way to compare prices before making a purchase.

## ✨ Features

- 📚 Browse popular books
- 💰 Compare prices across multiple retailers
- 🔍 Dynamic price range display
- 🛒 Direct purchase links to retailer websites
- 👤 User Registration
- 🔐 Secure Login
- 📱 Responsive user interface
- ⚡ Simple and easy navigation

## ⚙️ How It Works

1. User visits the homepage.
2. Popular books are displayed with a price range.
3. User clicks **Compare Prices**.
4. Prices from Amazon, Flipkart, and Bookswagon are displayed.
5. User clicks **Buy Now** to visit the retailer's website.

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/pranavikantegari/ZYPP-Book-Price-Comparison.git
```

### Navigate to Project Folder

```bash
cd ZYPP-Book-Price-Comparison
```

### Start XAMPP

- Start Apache
- Start MySQL

### Import Database

Import the provided `zypp.sql` file into MySQL.

### Open the Project

```
http://localhost/zypp
```
